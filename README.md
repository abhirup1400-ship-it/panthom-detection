# PANTHOM-DETECTS // GLOBAL THREAT INTELLIGENCE

**By Abhirup Payne**

A professional, real-time threat intelligence platform for detecting scams, phishing, malware, APT activity, and Business Email Compromise (BEC).

## 🚀 Live Demo

> Deploy via GitHub Pages — see below.

## ✨ Features

- **Message / SMS Analysis** — Paste any suspicious text and instantly score it
- **Email Intelligence** — Analyze sender addresses and email bodies
- **Phone Number Intel** — Detect high-risk international numbers
- **URL Decomposer** — Deep URL scanning for typosquatting, suspicious TLDs, shorteners
- **File & Document Forensics** — PDF text extraction + content analysis
- **Malware Detection** — Signature-based scanning for executables and scripts
- **APT Detection** — Advanced Persistent Threat indicators
- **BEC Detection** — Business Email Compromise pattern matching
- **Demo Scam Library** — Pre-loaded real-world scam scenarios
- **Global Threat Intel** — Education on worldwide scam tactics

## 🛠 Tech Stack

- Vanilla HTML / CSS / JavaScript (single-file, zero dependencies)
- Three.js — 3D loading animation and background
- PDF.js — Client-side PDF text extraction
- CryptoJS — Integrity utilities
- Google Fonts — Orbitron, JetBrains Mono, Space Mono, Rajdhani

## 📦 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from a branch`
4. Select branch `main` (or `master`) and folder `/ (root)`
5. Click **Save** — your site will be live in ~60 seconds
